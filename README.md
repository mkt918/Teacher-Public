# 担任業務支援アプリ

**🔗 デモ・利用はこちら: [https://mkt918.github.io/Teacher/](https://mkt918.github.io/Teacher/)**

担任業務を効率化するための統合Webアプリケーションです。生徒名簿管理、個人メモ、席替え、保護者会調整など、日々の業務をサポートします。

## 🎯 主な機能

### 📚 ダッシュボード
各種ツールへの統合アクセス。ToDo・週間スケジュール・年間行事をGASクラウドと自動同期。

### 👥 生徒名簿管理
- 生徒の追加・編集・削除
- 番号順・五十音順ソート
- CSV一括インポート/エクスポート
- 検索・フィルター機能

### 📝 個人メモ（Obsidian風）
- 生徒ごとの個別メモ
- Markdown記法サポート
- タグ機能（#面談、#保護者連絡など）
- 全メモのテキスト一括出力（NotebookLM連携用）

### 🪑 席替えツール
- くじ引きモード（トランプ／番号札）
- 座席の自動配置・入れ替え
- 印刷レイアウト調整

### 📅 保護者会調整ツール
- 希望調査集計・自動スケジュール調整・印刷

### 📅 時間割・出欠管理
- クラス時間割／自分の時間割の切り替え
- ドラッグ＆ドロップによる直感的な時間割編集
- 科目別授業時数の集計
- 生徒ごとの出欠記録（コマ単位）・警告表示
- **出欠サマリーの科目名をクリック → その科目の全授業日・曜日・時限を一覧表示**

### ✏️ テスト問題補助ツール
- 大問・設問の構造化作成
- 評価観点ごとの配点集計
- **問題一覧をNo順 / ランダム順で表示切り替え**
- **解答コピーを「記号で解答」「言葉で解答」「両方で解答」の3モードで選択**
- A4/B4 PDF出力
- 用語問題テンプレート生成

### 💾 保存システム
- リアルタイムセーブ（3スロット、自動保存）
- ステートセーブ（3スロット、手動保存）
- **GASクラウド同期**（起動時に自動同期、データ消失対策）

## 🚀 使い方

### 起動方法

1. `index.html` をブラウザで開く
2. または、ローカルサーバーで起動:

   ```powershell
   python -m http.server 8000
   # ブラウザで http://localhost:8000 にアクセス
   ```

### 基本的な使い方

1. **生徒名簿の登録**
   - 「生徒名簿」ページで「➕ 生徒追加」ボタンをクリック
   - 番号（4桁）、名前（漢字）、名前（ふりがな）を入力

2. **個人メモの記録**
   - 「個人メモ」ページで生徒を選択してMarkdown形式で記入
   - タグ（#面談など）で分類、「📤 全メモ出力」でエクスポート

3. **時間割の管理**
   - 「各種設定」→「時間割」でベースの時間割を設定
   - ドラッグ＆ドロップで科目を配置

4. **出欠の確認**
   - 出欠サマリーの科目名をクリックすると、その科目の授業一覧（日付・曜日・時限）を表示

5. **テスト問題の作成**
   - 「テスト問題補助」で大問・設問を入力
   - プレビューの「🔀」ボタンでランダム順に切り替え
   - 解答は「記号」「言葉」「両方」の3種類でコピー可能

6. **データの保存**
   - リアルタイムセーブ: データ変更後1秒で自動保存
   - GASクラウド同期: 起動時・変更時に自動でGoogleスプレッドシートへ同期

## 💾 データ保存について

### SafariのITP対策
SafariはITP（Intelligent Tracking Prevention）により、約7日間アクセスがないとLocalStorageを自動削除します。本アプリはGASクラウド同期を常時有効化することでこの問題に対処しています。

### 推奨運用
1. 定期的にステートセーブを実行
2. 重要なデータはCSV/テキスト出力でバックアップ
