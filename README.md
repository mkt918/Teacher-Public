# 担任業務支援アプリ

**🔗 デモ・利用はこちら: [https://mkt918.github.io/Teacher/](https://mkt918.github.io/Teacher/)**

担任業務を効率化するための統合Webアプリケーションです。生徒名簿管理、個人メモ、時間割・出欠、席替え、バス座席、保護者会調整、クラス係など、日々の業務をサポートします。

最新のアップデート内容は [PATCHNOTES.md](PATCHNOTES.md) を参照してください。

---

## 🎯 主な機能

### 📚 ダッシュボード
週間スケジュール・ToDoリストを表示する起点ページ。ToDo・週間スケジュール・年間行事はGASクラウドと自動同期。

### 👥 生徒名簿管理
- 生徒の追加・編集・削除
- **番号を指定して複数人を一括登録**（開始番号＋人数を指定するだけ。名前は後から入力）
- 番号順・五十音順ソート
- CSV一括インポート/エクスポート
- 検索・フィルター機能

### 📝 個人メモ（Obsidian風）
- 生徒ごとの個別メモ
- Markdown記法サポート
- タグ機能（#面談、#保護者連絡など）
- 全メモのテキスト一括出力（NotebookLM連携用）

### 📅 時間割・出欠管理
- クラス時間割／自分の時間割の切り替え
- ドラッグ＆ドロップによる直感的な時間割編集
- 科目名を変更すると時間割・当日の変更履歴にも自動で遡って反映
- 科目別授業時数の集計、**科目名クリックでその科目の全授業日・曜日・時限を一覧表示**
- 授業担当統計は期間指定で集計（デフォルトは今年度4月1日〜今日）
- 生徒ごとの出欠記録を**日付×時限の一覧表**で表示・入力
  - 各コマに科目名を常時表示、セルクリックで連続して欠席を確定
  - 「全欠席」ワンクリックボタンで1日まとめて登録も可能
- 時間割印刷（区切りの太線・登校前/放課後行つき）

### 🪑 席替えツール
- ①座席の形を決める → ②割り当てる → ③保存・印刷する、のステップガイド付き
- くじ引きモード（トランプ／番号札）と通常モードをカード型タブで切替
- ランダム配置・出席番号順配置、座席の🔒ロック機能
- 保存履歴の管理、印刷レイアウト調整

### 🚌 バス座席表
- フロントガラス・運転席・乗降口・タイヤなど、実物のバスに近い見た目
- 複数バスの管理、座席の自動ランダム配置
- **空席ロック／生徒ロック機能**（🔓/🔒）でランダム配置時に固定したい席を指定可能
- 印刷レイアウト対応

### 📅 保護者会時間決定ツール
- ①日程設定 → ②希望収集 → ③自動配置・確認 → ④保存・印刷、のステップガイド付き
- 生徒ごとに参加可能な時間帯を選択し希望として登録（進捗表示つき）
- **自動配置は「希望枠が少ない生徒」を優先して確定する方式**（デッドロックしにくい）
- 配置後に希望どおりかを一括チェック可能
- ボタン・文字サイズを拡大したユニバーサルデザインUI

### 📋 クラス係・年間行事予定・要録所見
- クラス係の割り当て・保存履歴管理
- 年間行事のカレンダー管理、CSV入出力
- 要録所見の下書き作成支援

### ✏️ テスト問題補助ツール
- 大問・設問の構造化作成、評価観点ごとの配点集計
- 問題一覧を**No順／ランダム順**で切り替え表示
- 解答コピーを**「記号で解答」「言葉で解答」「両方で解答」**の3モードで選択
- A4/B4 PDF出力
- 用語問題テンプレート生成（五十音順の語群作成など）

### 💾 保存システム
- リアルタイムセーブ（自動保存）／ステートセーブ（手動保存）
- **GASクラウド同期**：起動時から常時有効。Google スプレッドシートへ自動バックアップされ、Safari の ITP（約7日でLocalStorageが消える仕様）による消失を防止

## 🚀 使い方

### 起動方法

1. `index.html` をブラウザで開く
2. または、ローカルサーバーで起動:

   ```powershell
   python -m http.server 8000
   # ブラウザで http://localhost:8000 にアクセス
   ```

### 基本的な使い方

1. **生徒名簿の登録**
   - 「生徒名簿」ページで「➕ 生徒追加」、または「🔢 番号で一括登録」で開始番号と人数を指定
   - 番号（4桁）、名前（漢字）、名前（ふりがな）を入力

2. **個人メモの記録**
   - 「個人メモ」ページで生徒を選択してMarkdown形式で記入
   - タグ（#面談など）で分類、「📤 全メモ出力」でエクスポート

3. **時間割・出欠の管理**
   - 「各種設定」→「時間割」でベースの時間割を設定、ドラッグ＆ドロップで科目を配置
   - 出欠サマリーで生徒名をクリックすると、日付×時限の一覧表で出欠を入力可能
   - 科目名をクリックすると、その科目の全授業日一覧を確認可能

4. **席替え・バス座席・保護者会**
   - それぞれのページ上部の「❓ 使い方」ボタンでステップガイドを確認できます
   - 初回訪問時はガイドが自動で開きます

5. **テスト問題の作成**
   - 「テスト補助」で大問・設問を入力
   - プレビューの「🔀」ボタンでランダム順に切り替え
   - 解答は「記号」「言葉」「両方」の3種類でコピー可能

6. **データの保存**
   - リアルタイムセーブ: データ変更後1秒で自動保存
   - GASクラウド同期: 起動時・変更時に自動でGoogleスプレッドシートへ同期

## 💾 データ保存について

### SafariのITP対策
SafariはITP（Intelligent Tracking Prevention）により、約7日間アクセスがないとLocalStorageを自動削除します。本アプリはGASクラウド同期を常時有効化することでこの問題に対処しています。

### 推奨運用
1. 定期的にステートセーブを実行
2. 重要なデータはCSV/テキスト出力でバックアップ
